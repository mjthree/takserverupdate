# TAK Update Builder

A standalone script to generate `product.inf` and `product.infz` files for ATAK update servers.

## Quick Start

1. Navigate to the `tak-update-builder` folder
2. Run `npm install` to install dependencies
3. Run `node build-update.js [folder_path]` or use `build-update.bat` on Windows

See `tak-update-builder/README.md` for full documentation.
